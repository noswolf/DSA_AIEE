Linear & Binary Search

Finding data efficiently is a fundamental operation. In this module, we compare Linear Search (O(N)) and Binary Search (O(log N)), demonstrating the massive impact of order and divide-and-conquer strategies.

Linear Search O(n) Binary Search O(log n) Divide-and-Conquer

Sorting Algorithms

Sorting structures data in order to make it searchable and organized. We analyze four core sorting strategies: Bubble Sort, Selection Sort, Insertion Sort, and Merge Sort.

Bubble Sort Selection Sort Insertion Sort Merge Sort

Hash Tables & Collision Resolution

Hash Tables map keys directly to indices for O(1) constant-time search. When keys map to the same index, a Collision occurs. We explore collision resolution via Linear Probing.

Direct Lookup O(1) Linear Probing Collision Resolution
